# Cisco Boot Camp: AWS Cloud Computing & Check Point Security Administration

A comprehensive 7-day intensive bootcamp integrating enterprise networking, cloud infrastructure, and cybersecurity administration.

## Overview

Master the convergence of Cisco networking, AWS cloud services, and Check Point security. This hands-on course teaches you to architect, deploy, and secure hybrid enterprise networks spanning on-premises data centers and cloud environments.

## Course Curriculum

| Day | Topic | Duration | Key Skills |
|-----|-------|----------|-----------|
| 1 | Networking & Cloud Fundamentals | 6 hrs | OSI model, cloud concepts, AWS basics |
| 2 | Network Configuration & AWS Compute | 7 hrs | Interface config, EC2 deployment |
| 3 | LAN Switching & AWS Networking | 6 hrs | Switching, VLANs, VPC design |
| 4 | Routing & AWS Storage Solutions | 6 hrs | BGP/OSPF, S3, load balancing |
| 5 | Network Security & IAM | 7 hrs | ACLs, VPNs, security groups, IAM policies |
| 6 | Advanced Networking & Containers | 7 hrs | SDN, Docker, network automation |
| 7 | Cybersecurity & Check Point Admin | 6 hrs | Firewall rules, threat prevention, compliance |

## Core Competencies

- **Networking Architecture**: OSI/TCP-IP models, IP subnetting, VLAN design, routing protocols (OSPF, BGP)
- **AWS Services**: EC2, VPC, S3, EBS, RDS, CloudWatch, auto-scaling
- **Security Implementation**: firewalls, VPNs, NAT, IAM policies, Security Groups, Check Point management
- **Modern Infrastructure**: containerization, SDN principles, network automation tools

## Industry Certifications

- Cisco CCNA Routing & Switching
- AWS Solutions Architect Associate/Professional
- Check Point Certified Security Administrator (CCSA)
